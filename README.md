# Phone Store
> An app where you can find your needs (Phone is just a name, you can find other thing too!)

# Usage

1. Clone this repo
```
git clone https://github.com/aldobangun/phone-store.git
```

2. Go to app folder
```
cd phone-store
```

3. install packages
```
npm install
```

4. run the app
```
npm start
```
